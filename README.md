# 👋 Olá, eu sou o Heitor Borges

Sou desenvolvedor full stack e empreendedor, fundador das empresas **Pet Optimizer Tecnologia LTDA** e **Syntexa Code Tecnologia LTDA**.

Atuo no desenvolvimento de sistemas web completos, com foco em **produtos SaaS**, **sistemas de gestão**, **arquitetura escalável** e **integrações com serviços governamentais e terceiros**.

Grande parte dos projetos que desenvolvo são **privados e proprietários**, utilizados por empresas reais em produção.  
Este GitHub tem como objetivo apresentar **minha abordagem técnica, organização de código, arquitetura e padrões de desenvolvimento**.
